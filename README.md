### ** Mini-Project: Autonomous Vehicle Lane Detection**



Autonomous vehicle lane detection for perception in self-driving systems.
Implements gradient magnitude & direction computation from road frames using classical edge cues and lane markings.
Applies Sobel Operator and Canny Edge Detector to extract lane boundaries and object edges.
Highlights gradient computation as the core foundation for lane and obstacle perception in autonomous tasks.
